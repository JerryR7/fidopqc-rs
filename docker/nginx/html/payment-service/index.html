Payment Service

This is the Payment Service for the PasskeyMesh Gateway microservices demo.

The API endpoint is available at /api.

Service Information

Service Name: Payment Service

TLS Support: Traditional TLS only (Non-PQC)

PQC Access: Via Quantum-Safe-Proxy Sidecar

Authentication: JWT validation

Features: Payment processing and transaction history

⚠️ Security Notice

This service uses traditional TLS and requires a Quantum-Safe-Proxy sidecar for PQC protection.

In a real-world scenario, this represents a legacy service that cannot be directly upgraded to support PQC.
